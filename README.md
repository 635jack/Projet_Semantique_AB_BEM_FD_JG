# Projet Knowledge Extraction from Unstructured Text

**Université Paris Cité - Master 2 VMI**  
**Cours :** IFLCE085 Recherche et extraction sémantique à partir de texte  
**Enseignante :** Salima Benbernou

## Description
Ce projet vise à extraire des connaissances à partir de textes non structurés en utilisant le dataset **SciREX**.

## Structure du Repository
- `release_data/` : Dataset SciREX (train, dev, test).
- `preprocessing.py` : Module contenant les fonctions de nettoyage et de vectorisation.
- `PartieA_Preprocessing.ipynb` : Notebook principal pour l'exécution de la Partie A.
- `requirements.txt` : Dépendances Python.
